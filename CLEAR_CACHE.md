# Clear Browser Cache to See New Design

The website has been completely redesigned with a modern, trendy UX! To see the changes, you need to clear your browser cache.

## Quick Methods to See the New Design:

### Method 1: Hard Refresh (Fastest)
**Windows/Linux:**
- Chrome/Edge: `Ctrl + Shift + R` or `Ctrl + F5`
- Firefox: `Ctrl + Shift + R` or `Ctrl + F5`

**Mac:**
- Chrome/Edge: `Cmd + Shift + R`
- Firefox: `Cmd + Shift + R`
- Safari: `Cmd + Option + R`

### Method 2: Open in Incognito/Private Window
- Chrome: `Ctrl/Cmd + Shift + N`
- Firefox: `Ctrl/Cmd + Shift + P`
- Safari: `Cmd + Shift + N`

### Method 3: Clear Browser Cache
**Chrome:**
1. Press `Ctrl/Cmd + Shift + Delete`
2. Select "Cached images and files"
3. Click "Clear data"

**Firefox:**
1. Press `Ctrl/Cmd + Shift + Delete`
2. Select "Cache"
3. Click "Clear Now"

**Safari:**
1. Go to Safari > Preferences > Advanced
2. Enable "Show Develop menu"
3. Develop > Empty Caches

### Method 4: Developer Tools
1. Open Developer Tools (`F12`)
2. Right-click the refresh button
3. Select "Empty Cache and Hard Reload"
